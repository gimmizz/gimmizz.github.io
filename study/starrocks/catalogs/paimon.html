6-7. Paimon Catalog

📌 한 줄 요약

Apache Paimon은 Flink Stream → Lake 형식. StarRocks는 v3.1부터 Paimon 테이블 SELECT를 지원한다.

PROPERTIES — Filesystem 카탈로그

CREATE EXTERNAL CATALOG paimon_fs
PROPERTIES (
  "type"                = "paimon",
  "paimon.catalog.type" = "filesystem",
  "paimon.catalog.warehouse" = "s3://paimon/warehouse",
  "aws.s3.endpoint"     = "https://minio.internal:9000",
  "aws.s3.access_key"   = "...",
  "aws.s3.secret_key"   = "...",
  "aws.s3.enable_path_style_access" = "true"
);

# HDFS 위에 두는 경우
"paimon.catalog.warehouse" = "hdfs://nn/warehouse"

PROPERTIES — Hive 카탈로그

CREATE EXTERNAL CATALOG paimon_hms
PROPERTIES (
  "type"                = "paimon",
  "paimon.catalog.type" = "hive",
  "hive.metastore.uris" = "thrift://hms.example.com:9083",
  "paimon.catalog.warehouse" = "hdfs://nn/warehouse"
);

활용

  • Snapshot / Time Travel 일부 지원.
  • Stream 적재가 Flink Paimon 측에서 일어나고, StarRocks는 분석 가속기 역할.
  • StarRocks에서 Paimon 쓰기는 미지원.
클러스터 적용 노트

이미 Flink CDC + Paimon으로 실시간 레이크하우스를 구축 중이라면, StarRocks를 별도 OLAP 적재 없이 직접 분석 엔진으로 붙이는 패턴이 효과적입니다.
