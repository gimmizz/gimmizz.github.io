6-10. MaxCompute Catalog

📌 한 줄 요약

Alibaba Cloud MaxCompute(ODPS)의 테이블에 직접 SELECT 가능. 알리바바 클라우드 환경에서 주로 쓰임.

PROPERTIES

CREATE EXTERNAL CATALOG mc_prod
PROPERTIES (
  "type"                  = "maxcompute",
  "aliyun.odps.endpoint"  = "http://service.cn-hangzhou.maxcompute.aliyun.com/api",
  "aliyun.odps.project"   = "your_project",
  "aliyun.odps.access.id" = "...",
  "aliyun.odps.access.key" = "...",
  "aliyun.odps.tunnel.endpoint" = "http://dt.cn-hangzhou.maxcompute.aliyun.com"
);

주의사항

  • 알리바바 클라우드 외부에서 사용 시 네트워크 비용/지연이 큼.
  • StarRocks에서 MaxCompute 쓰기는 미지원.
  • 대용량 풀스캔은 MaxCompute의 슬롯 비용을 폭증시킬 수 있음.
클러스터 적용 노트

국내 자체 데이터센터 환경에서는 잘 쓰이지 않음. 알리바바 클라우드 의존이 있는 일부 데이터를 결합 분석할 때만 검토.
