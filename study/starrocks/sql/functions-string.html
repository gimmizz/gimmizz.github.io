SQL-4. 문자열 함수

기본

함수예시
LENGTH(s) / CHAR_LENGTH(s)바이트 / 문자 길이
UPPER(s) / LOWER(s)대문자/소문자
TRIM/LTRIM/RTRIM(s)양쪽/왼쪽/오른쪽 공백 제거
CONCAT(a, b, ...)이어붙이기
CONCAT_WS(sep, a, b, ...)구분자 사용
REPLACE(s, from, to)치환
REVERSE(s)뒤집기
SUBSTR(s, start[, len]) / SUBSTRING부분 문자열 (start는 1부터)
LEFT(s, n) / RIGHT(s, n)앞/뒤 n자
LPAD/RPAD(s, len, pad)패딩

검색

SELECT INSTR('hello world', 'world');     -- 7 (1-base, 0이면 없음)
SELECT POSITION('w' IN 'hello world');
SELECT LOCATE('w', 'hello world');
SELECT 'hello world' LIKE '%world';
SELECT 'hello world' LIKE 'h_llo%';

정규식

SELECT REGEXP_LIKE(col, '^[A-Z]+$');
SELECT col REGEXP '^[0-9]+$';
SELECT REGEXP_REPLACE(col, '\\s+', ' ');
SELECT REGEXP_EXTRACT(url, '/user/([0-9]+)/', 1);  -- 그룹 1
SELECT REGEXP_EXTRACT_ALL('a1b2c3', '[a-z]([0-9])', 1);  -- ['1','2','3']

분할 / 결합

SELECT SPLIT('a,b,c', ',');               -- ['a','b','c']
SELECT SPLIT_PART('a,b,c', ',', 2);       -- 'b' (1-base)
SELECT ARRAY_JOIN(ARRAY['a','b','c'], '-'); -- 'a-b-c'

JSON 추출 (간이)

SELECT GET_JSON_STRING(payload, '$.user.id');
SELECT GET_JSON_INT   (payload, '$.amount');
SELECT GET_JSON_DOUBLE(payload, '$.price');
-- 본격적인 JSON 처리는 SQL-7. JSON 페이지 참조

인코딩

SELECT TO_BASE64('hello');
SELECT FROM_BASE64('aGVsbG8=');
SELECT URL_ENCODE('a b/c?d=1');
SELECT URL_DECODE('a%20b');
SELECT MD5(s);
SELECT SHA2(s, 256);
SELECT HEX(123);  -- '7B'

문자열 ↔ 숫자

SELECT CAST('123' AS BIGINT);
SELECT CAST('1.5' AS DOUBLE);
SELECT FORMAT(12345.678, 2);  -- '12,345.68'
SELECT NULLIF(s, '');         -- 빈 문자열을 NULL로
클러스터 적용 노트
  • LIKE '%foo%'가 잦은 컬럼은 N-Gram Bloom 인덱스(3-2) 추가 검토.
  • 운영 로그/메시지 분석에서 정규식이 무겁게 도는 경우, ETL 단계에서 미리 split/extract해 컬럼화하는 게 SR 부하를 크게 줄임.
