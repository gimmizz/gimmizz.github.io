SQL-3. 날짜·시간 함수

현재 시각

함수의미
NOW() / CURRENT_TIMESTAMP()현재 DATETIME
CURDATE() / CURRENT_DATE()현재 DATE
UTC_TIMESTAMP()UTC 시각
UNIX_TIMESTAMP([dt])유닉스 epoch 초

truncate / 형변환

-- 핵심: date_trunc는 파티션 키와 같이 가장 자주 쓰는 함수
SELECT date_trunc('day',  now());     -- 2026-05-07 00:00:00
SELECT date_trunc('hour', now());
SELECT date_trunc('month', now());
SELECT date_trunc('week', now());     -- 월요일 시작

-- 형변환
SELECT CAST('2026-05-07' AS DATE);
SELECT CAST('2026-05-07 10:00:00' AS DATETIME);
SELECT FROM_UNIXTIME(1746547200);
SELECT UNIX_TIMESTAMP('2026-05-07 10:00:00');

차이 / 가감

SELECT DATEDIFF('2026-05-07','2026-05-01');        -- 6 (일 차이)
SELECT DATE_DIFF('hour', t1, t2);                  -- 단위 명시
SELECT TIMESTAMPDIFF(SECOND, t1, t2);
SELECT DATE_ADD('2026-05-07', INTERVAL 7 DAY);
SELECT DATE_SUB(now(), INTERVAL 1 HOUR);
SELECT now() + INTERVAL 30 MINUTE;

포맷

SELECT DATE_FORMAT(now(), '%Y-%m-%d %H:%i:%s');     -- 2026-05-07 10:00:00
SELECT DATE_FORMAT(now(), '%Y%m%d');                -- 20260507
SELECT STR_TO_DATE('07/05/2026','%d/%m/%Y');

추출

함수의미
YEAR(dt) / MONTH(dt) / DAY(dt)년/월/일
HOUR(dt) / MINUTE(dt) / SECOND(dt)시/분/초
DAYOFWEEK(dt)1=일 ... 7=토 (MySQL 호환)
DAYOFYEAR(dt)해의 며칠째
WEEK(dt)해의 몇 번째 주

기간 윈도

-- 마지막 7일
WHERE event_time >= now() - INTERVAL 7 DAY

-- 지난 달
WHERE date_trunc('month', event_time) = date_trunc('month', now() - INTERVAL 1 MONTH)

-- 시간대 변환 (UTC ↔ 로컬)
SELECT CONVERT_TZ('2026-05-07 10:00:00', '+00:00', '+09:00');
클러스터 적용 노트
  • 파티션 키는 거의 항상 date_trunc('day', ...) 또는 date_trunc('hour', ...). 시간대는 적재 시점에 통일(보통 KST 또는 UTC).
  • FROM_UNIXTIME·UNIX_TIMESTAMP는 epoch 기반 적재(예: 모바일 로그)에서 자주 사용. 적재 시 DATETIME으로 통일하면 쿼리가 단순해짐.
  • 대시보드의 "오늘" 필터는 WHERE dt = CURDATE()가 가장 빠름 — 함수 적용 없이 파티션 키와 일치.
