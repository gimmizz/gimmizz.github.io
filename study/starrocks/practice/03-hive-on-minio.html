P-3. Hive Catalog + MinIO 로컬 실습

📌 목표

로컬에 MinIO + HMS + StarRocks를 띄우고 Hive Catalog로 외부 테이블을 SELECT 한다. 운영 PM 클러스터에서 그대로 옮겨갈 수 있는 패턴.

네트워크 + MinIO 시작 
docker network create srnet

docker run -d --name minio --network srnet \
  -p 9000:9000 -p 9001:9001 \
  -e MINIO_ROOT_USER=minio -e MINIO_ROOT_PASSWORD=minio123 \
  minio/minio server /data --console-address ":9001"

# 버킷 생성 (mc 또는 콘솔 http://localhost:9001)
docker run --rm --network srnet --entrypoint sh minio/mc -c \
  "mc alias set m http://minio:9000 minio minio123 && mc mb m/warehouse"
HMS 시작 (Postgres + Hive Metastore) 
docker run -d --name hms-pg --network srnet \
  -e POSTGRES_USER=hive -e POSTGRES_PASSWORD=hive -e POSTGRES_DB=metastore \
  postgres:14

docker run -d --name hms --network srnet -p 9083:9083 \
  -e DATABASE_HOST=hms-pg -e DATABASE_DB=metastore \
  -e DATABASE_USER=hive -e DATABASE_PASSWORD=hive \
  -e S3_ENDPOINT=http://minio:9000 \
  -e S3_ACCESS_KEY=minio -e S3_SECRET_KEY=minio123 \
  -e S3_PATH_STYLE_ACCESS=true \
  apache/hive:3.1.3 /opt/hive/bin/hive --service metastore

(이미지/환경 변수는 실제 도구에 맞게 조정. 운영에서는 hive-site.xml를 직접 마운트하는 경우가 많음.)

StarRocks Allin1 시작 
docker run -d --name sr --network srnet \
  -p 9030:9030 -p 8030:8030 -p 8040:8040 \
  starrocks/allin1-ubuntu:latest
StarRocks에서 Hive Catalog 생성 
mysql -h 127.0.0.1 -P 9030 -u root

CREATE EXTERNAL CATALOG hive_local
PROPERTIES (
  "type" = "hive",
  "hive.metastore.type" = "hive",
  "hive.metastore.uris" = "thrift://hms:9083",
  "aws.s3.endpoint"     = "http://minio:9000",
  "aws.s3.access_key"   = "minio",
  "aws.s3.secret_key"   = "minio123",
  "aws.s3.enable_path_style_access" = "true"
);
Hive 테이블 생성 + 데이터 INSERT (StarRocks에서!) 
SET CATALOG hive_local;
CREATE DATABASE dwh;

CREATE TABLE dwh.events (
  event_time DATETIME,
  user_id    BIGINT,
  event_type VARCHAR(32),
  dt         DATE
)
PARTITION BY (dt)
PROPERTIES (
  "location"     = "s3://warehouse/dwh.db/events/",
  "file_format"  = "parquet"
);

INSERT INTO dwh.events VALUES
  ('2026-05-07 10:00:00', 100, 'view',  '2026-05-07'),
  ('2026-05-07 10:05:00', 100, 'click', '2026-05-07'),
  ('2026-05-07 10:10:00', 200, 'view',  '2026-05-07');

SELECT count(*) FROM dwh.events WHERE dt = '2026-05-07';
외부 카탈로그와 내부 테이블 JOIN 
SET CATALOG default_catalog;
USE shop;  -- P-2에서 만든 테이블

SELECT u.user_id, count(*) cnt
FROM hive_local.dwh.events e
LEFT JOIN shop.orders o ON e.user_id = o.user_id
GROUP BY u.user_id;
클러스터 적용 노트

이 실습 그대로 운영 PM 환경에 옮길 때는 (1) MinIO → 운영 S3/HDFS, (2) HMS → 운영 HMS, (3) StarRocks Allin1 → 약 20대 PM 분산 클러스터로만 바꾸면 됩니다. 카탈로그 등록 SQL은 동일.
