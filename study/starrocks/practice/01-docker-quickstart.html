P-1. 노트북에서 5분 만에 띄우기

📌 목표

로컬 노트북에서 Docker로 단일 노드 StarRocks를 띄우고, MySQL CLI로 첫 SQL을 실행한다.

준비물

  • Docker Desktop (Mac/Windows) 또는 Docker Engine (Linux)
  • MySQL CLI 또는 DBeaver
  • 여유 메모리 ≥ 4GB
StarRocks Allin1 컨테이너 실행 
docker run -p 9030:9030 -p 8030:8030 -p 8040:8040 \
  -itd --name sr-quickstart \
  starrocks/allin1-ubuntu:latest
FE가 ready 될 때까지 대기 (~30초) 
docker logs -f sr-quickstart  # "FE started" 비슷한 로그 확인 후 Ctrl-C
MySQL CLI로 접속 
mysql -h 127.0.0.1 -P 9030 -u root -p
# (비밀번호 없음 — Enter)
첫 데이터베이스/테이블 만들기 
CREATE DATABASE demo;
USE demo;

CREATE TABLE quick_test (
  id   BIGINT NOT NULL,
  name VARCHAR(32),
  ts   DATETIME
)
PRIMARY KEY(id)
DISTRIBUTED BY HASH(id);

INSERT INTO quick_test VALUES
  (1, 'alice', NOW()),
  (2, 'bob',   NOW()),
  (3, 'carol', NOW());

SELECT * FROM quick_test;
SHOW BACKENDS\G
SHOW FRONTENDS\G
웹 UI 확인
  • FE: http://127.0.0.1:8030
  • BE: http://127.0.0.1:8040
정리 
docker stop sr-quickstart && docker rm sr-quickstart
클러스터 적용 노트

Allin1 이미지는 학습/실험 전용. 운영 클러스터는 4장 배포의 PM 절차로 구축. Allin1로 익힌 SQL/문법은 그대로 운영에서도 통하므로, 새 사용자는 이 단계부터 시작하는 게 좋음.
