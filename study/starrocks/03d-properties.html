3-3. CREATE TABLE PROPERTIES 레퍼런스

📌 한 줄 요약

테이블 동작·저장·인덱스·MV 모두 PROPERTIES로 제어한다. 자주 쓰는 20여 개를 카테고리별로 정리.

스토리지 / 복제

키값설명
replication_num3 (기본)Tablet 복제본 수. 약 17 BE면 3 권장.
storage_mediumSSD / HDD핫 데이터 매체. SSD가 일반.
storage_cooldown_time"2026-08-01 00:00:00"이 시점 이후 콜드 매체로 이동.
storage_volume볼륨 이름Shared-data 모드에서 객체 스토리지 볼륨 지정.
compressionLZ4 (기본) / ZSTD / SNAPPY / ZLIB컬럼 압축 코덱. ZSTD가 압축률 우수.

파티션 / 버킷

키설명
dynamic_partition.enabletrue/false
dynamic_partition.time_unitHOUR/DAY/MONTH
dynamic_partition.start / .end보존/미래 파티션 범위
dynamic_partition.prefix파티션 이름 접두어 (보통 "p")
dynamic_partition.buckets새 파티션의 버킷 수
colocate_withColocate Group 이름. 같은 그룹 테이블 간 셔플 없는 JOIN.

인덱스

키설명
bloom_filter_columns"col1, col2" — 등호 lookup 가속

Primary Key 전용

키값설명
enable_persistent_indextruePK 인덱스 일부를 디스크로. 대용량 PK 필수.
persistent_index_typeLOCAL / CLOUD_NATIVEv3.x — 영속 인덱스 종류.
write_quorumMAJORITY (기본) / ALL / ONETablet 쓰기 quorum 정책.

Materialized View

키설명
enable_query_rewriteMV 자동 재작성 허용 여부
partition_refresh_number한 번에 리프레시할 파티션 수
excluded_trigger_tables이 테이블 변경은 MV 트리거 X
session.query_timeout리프레시 잡의 타임아웃

종합 예시

CREATE TABLE mart.orders (
  order_id   BIGINT NOT NULL,
  user_id    BIGINT NOT NULL,
  status     VARCHAR(32),
  amount     DECIMAL(18,2),
  updated_at DATETIME
)
PRIMARY KEY(order_id)
PARTITION BY date_trunc('day', updated_at)
DISTRIBUTED BY HASH(order_id)
ORDER BY(updated_at)
PROPERTIES (
  "replication_num"          = "3",
  "storage_medium"           = "SSD",
  "compression"              = "ZSTD",
  "enable_persistent_index"  = "true",
  "bloom_filter_columns"     = "user_id",
  "dynamic_partition.enable" = "true",
  "dynamic_partition.time_unit" = "DAY",
  "dynamic_partition.start"  = "-90",
  "dynamic_partition.end"    = "3",
  "dynamic_partition.prefix" = "p"
);
클러스터 적용 노트
  • 표준 PROPERTIES 템플릿을 팀 위키로 정리해 놓고, 모든 신규 테이블이 같은 기본값을 갖게 하면 운영 부담이 크게 줄어듭니다.
  • compression=ZSTD: SSD가 충분하니 압축률보다 디코딩 비용이 살짝 더 들어도 OK인 워크로드에서 좋음. 핫 테이블은 LZ4가 안전.
  • storage_cooldown_time: 향후 SSD가 부족해지면 콜드 매체(별도 HDD/오브젝트 스토리지)로 자동 이동시키는 정책을 미리 설계해두는 게 좋음.
